- AYUDA DE LA APLICACIÓN

Estas son las funcionalidades de la APP:

Selecciona la opción que desees de los menús. Según el tipo de usuario podrás acceder a unas u otras partes de esta APP. Si accedes como Usuario Normal(Comprador - Inquilino) dispondrás de las opciones de compra y búsqueda de anuncios. Si accedes como Usuario Premium(Vendedor - Propietario) dispondrás de las opciones de venta y creación de anuncios. Están diseñadas para atender tus necesidades.

Si necesitas más información, consulta el manual de usuario.

Si notas algún problema en esta App, por favor, contacta con el administrador.
